# display_lvgl_demo

Dark-themed UI demo for the Waveshare ESP32-S3-Touch-LCD-2.8 board.  
Use this as a **starting point template** for new projects on this hardware.

---

## What it shows

### Splash screen (2.8 s)
- KNI logo (converted from PNG, 160×160 px)
- Animated teal loading bar
- Fade-in transition to main screen

### Main screen
```
┌─────────────────────────────────┐  ← status bar (KNI MAKERS | ESP32-S3)
│                                 │
│   [content panel — scrollable]  │  ← 234 px tall, vertical scroll
│                                 │
├──────────┬──────────┬───────────┤  ← menu bar
│   HOME   │   INFO   │    SYS    │
└──────────┴──────────┴───────────┘
```

**HOME tab** — live system stats updated every second:
- Uptime (HH:MM:SS)
- Free heap (KB)
- CPU frequency

**INFO tab** — project description, scrollable text

**SYS tab** — hardware specification table (SoC, CPU, Flash, PSRAM, LCD, Touch, LVGL, IDF)

## Build & flash

```bash
cd examples/display_lvgl_demo

# First time only
idf.py set-target esp32s3

idf.py build
idf.py -p /dev/ttyACM0 flash monitor
```

Replace `/dev/ttyACM0` with your actual port (`ls /dev/ttyACM* /dev/ttyUSB*`).

## Using as a template for a new project

1. Copy this folder to your new project location
2. Update `CMakeLists.txt` at the root if needed
3. Replace `kni_logo.c / kni_logo.h` with your own image (see `tools/png2lvgl.py`)
4. Edit `main.c` — the UI is split into clean functions per tab:
   - `build_splash_screen()` — splash screen
   - `build_home_panel()` — HOME tab content
   - `build_info_panel()` — INFO tab content
   - `build_sys_panel()` — SYS tab content
5. Add your custom content — the scroll, theming, and tab switching are already wired up

### Color palette (edit at top of `main.c`)

```c
#define C_BG       0x07111A   // background (very dark navy)
#define C_CARD     0x16263A   // card surface
#define C_TEAL     0x00CCA3   // accent — matches KNI logo
#define C_TEXT     0xDDECF8   // primary text
#define C_SUBTEXT  0x6888A8   // secondary text
```

## sdkconfig.defaults

Key settings applied automatically on first build:

| Setting | Value | Reason |
|---------|-------|--------|
| Flash | 16 MB QIO | Waveshare 27690 hardware |
| PSRAM | Octal 80 MHz | 8 MB R8 module |
| CPU | 240 MHz | Maximum performance |
| FreeRTOS tick | 1 kHz | Smooth LVGL input timing |
| LVGL color depth | 16-bit | ST7789 native format |
| Montserrat 14 | enabled | Default UI font |
| Montserrat 24 | enabled | Splash screen title |
