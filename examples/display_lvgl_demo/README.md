# display_lvgl_demo

Dark-themed UI demo for the Waveshare ESP32-S3-Touch-LCD-2.8 board.  
Shows how to use the `kni_ui` component — this file is your **starting template**.

---

## What it shows

### Splash screen (2.8 s)
- KNI logo (converted from PNG, 160×160 px)
- Animated teal loading bar
- Fade-in transition to main screen

### Main screen
```
┌─────────────────────────────────┐  ← status bar (KNI MAKERS | ESP32-S3)
│                                 │
│   [content panel — scrollable]  │  ← 234 px tall, vertical scroll
│                                 │
├──────────┬──────────┬───────────┤  ← menu bar
│   HOME   │   INFO   │    SYS    │
└──────────┴──────────┴───────────┘
```

**HOME tab** — live system stats updated every second (uptime, free heap, CPU)  
**INFO tab** — project description, scrollable wrapped text  
**SYS tab** — hardware spec table (SoC, CPU, Flash, PSRAM, LCD, Touch, LVGL, IDF)

## Build & flash

```bash
cd examples/display_lvgl_demo

# first time only
idf.py set-target esp32s3

idf.py build
idf.py -p /dev/ttyACM0 flash monitor
```

Replace `/dev/ttyACM0` with your actual port (`ls /dev/ttyACM* /dev/ttyUSB*`).

## Using as a template for a new project

1. Copy this folder to your new project
2. Replace `kni_logo.c / kni_logo.h` with your own image (`tools/png2lvgl.py`)
3. Edit `main.c` — define your tab builders and fill in `ui_cfg`:

```c
static const kni_ui_tab_t tabs[] = {
    { "HOME", build_home },
    { "INFO", build_info },
};

static const kni_ui_cfg_t ui_cfg = {
    .title     = "MY PROJECT",
    .subtitle  = "ESP32-S3",
    .logo      = &my_logo,
    .splash_ms = 2800,
    .tabs      = tabs,
    .tab_count = 2,
};
```

4. Inside each builder, use the `kni_ui_*` helpers:

```c
static void build_home(lv_obj_t *panel)
{
    lv_obj_t *col  = kni_ui_col(panel);           // for multiple stacked cards
    lv_obj_t *card = kni_ui_card(col, "STATUS");
    lv_obj_t *ip   = kni_ui_kv_row(card, "IP", "0.0.0.0");
    // update later: lv_label_set_text(ip, "192.168.1.1");
}
```

Full API: [`kni_ui/README.md`](../../kni_ui/README.md)

## sdkconfig.defaults

Key settings applied automatically on first build:

| Setting | Value | Reason |
|---|---|---|
| Flash | 16 MB QIO | Waveshare 27690 hardware |
| PSRAM | Octal 80 MHz | 8 MB R8 module |
| CPU | 240 MHz | Maximum performance |
| FreeRTOS tick | 1 kHz | Smooth LVGL input timing |
| LVGL color depth | 16-bit | ST7789 native format |
| Montserrat 14 | enabled | Default UI font |
| Montserrat 24 | enabled | Splash screen title |
